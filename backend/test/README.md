# Backend Test Assets

This folder contains test and verification scripts for the ContextPilot backend.

## Scripts
- `test-api.sh`: End-to-end API test suite.
- `pre-demo-check.sh`: Quick pre-demo smoke checks against a deployed or local API.
- `verify-tools.sh`: Verifies tool-calling flow for the configured AI provider.
- `check-env.sh`: Validates required environment variables are present.
- `test-anthropic-batch.sh`: Verifies Anthropic Message Batches API access.
